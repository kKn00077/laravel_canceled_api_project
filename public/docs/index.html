Introduction

쌀먹 API에 관련한 문서입니다.

Base URL

http://sm-dev.udnm.co.kr

Authenticating requests

아래의 엔드포인트 중에서
인증이 필요한 엔드포인트의 경우
헤더에 Authorization: "Bearer your-token"
데이터를 추가해주세요.

인증이 필요한 모든 엔드포인트는 해당 헤더값이 필수입니다.

Advertisement

광고에 관련한 API를 지원합니다.

광고 리스트

포지션(광고 위치)에 맞는 광고 리스트를 가져옵니다.

Example request:
curl --request GET \
    --get "http://sm-dev.udnm.co.kr/api/ad/list?position=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://sm-dev.udnm.co.kr/api/ad/list"
);

const params = {
    "position": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

토큰 생성 (로그인)

토큰 생성 API 입니다. 일반적으로 로그인 처리를 위한 토큰을 발급받을 때 사용할 수 있습니다.

Example request:
curl --request POST \
    "http://sm-dev.udnm.co.kr/api/auth/token" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"platform\": \"naver\",
    \"email\": \"test@test.co.kr\",
    \"device_name\": \"device-name\",
    \"password\": \"test1234\",
    \"id\": \"111086235715594204387\"
}"

const url = new URL(
    "http://sm-dev.udnm.co.kr/api/auth/token"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "platform": "naver",
    "email": "test@test.co.kr",
    "device_name": "device-name",
    "password": "test1234",
    "id": "111086235715594204387"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

curl --request DELETE \
    "http://sm-dev.udnm.co.kr/api/auth/token" \

    --header "Content-Type: application/json" \
    --header "Accept: application/json"

    "http://sm-dev.udnm.co.kr/api/auth/token"
);

const headers = {
    "Authorization": "Bearer f856ehcDPEaVvkab4g316Zd",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

    method: "DELETE",
    headers,
}).then(response => response.json());
